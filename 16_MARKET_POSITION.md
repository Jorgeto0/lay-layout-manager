# MARKET POSITION

## The Problem (Validated)
Users are frustrated that no app automatically saves and restores layouts
when connecting/disconnecting monitors on macOS. This is a real, active pain
point with no perfect solution today.

## Competitor Landscape

### Snapping tools only (no layout saving)
- **Rectangle** — free, open source, keyboard shortcuts, no layout saving
- **Magnet** — $4.99 App Store, drag-to-snap, no layout saving
- **MacTiler** — multi-window positioning, no layout saving

### Layout saving but limited automation
- **Moom** — $14.99, saves/restores layouts manually, Spaces support is weak,
  no automatic triggers on monitor change
- **Display Maid** — saves window positions per display profile, no Spaces support

### Closest competitor
- **Spencer** — saves/restores layouts across Spaces and Displays, launches apps,
  remembers number of Spaces. Direct download only (not App Store).
  Still requires manual save/restore triggers. In active beta as of mid-2025.

## The Gap We Fill
No App Store app that:
1. Automatically restores layouts on monitor change with zero user action
2. Correctly handles macOS Spaces
3. Is simple enough for non-power-users

## Our Differentiation (3 pillars)

### 1. Silent Automation
- Layout restores automatically when monitor connects/disconnects
- No button to press, no manual trigger
- It just works — like AirPods switching devices

### 2. Dead Simple UI
- Competitors are built for power users
- We target the normal Mac user who has an external monitor at their desk
- Onboarding: grant permission → done
- No complex profiles, no scripting, no learning curve

### 3. App Store Distribution
- Spencer and Display Maid are direct download only
- App Store = trust signal for mainstream users
- Easier discovery, easier purchase, automatic updates

## Target User
Mac user with at least one external monitor who is frustrated that their
windows scatter every time they dock or undock their MacBook.
Not a power user. Does not want to configure anything.

## Pricing Strategy
One-time purchase. $6.99–$9.99 range.
Below Moom ($14.99), competitive with Magnet ($4.99).
No subscription — subscriptions are a trust barrier for utility apps.

## Success Metric
User connects monitor → all windows are back where they were.
That is the entire value proposition.
